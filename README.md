# Light Years Game App

A 2D game application developed in modern C++ using SFML, CMake, and Git.

## Overview

Light Years Game App is a personal software engineering project focused on learning modern C++ development practices, game architecture, build systems, source control, and multimedia programming.

The project is being developed incrementally using industry-standard tools and workflows.

## Technologies

* C++17
* SFML
* CMake
* Git
* GitHub
* Visual Studio 2022

## Project Structure

```text
light-years-game-app/
├── LightYearsEngine/
├── LightYearsGame/
├── build/
└── CMakeLists.txt
```

## Current Features

* CMake project configuration
* Multi-project architecture
* Visual Studio 2022 integration
* Source control with Git and GitHub

## Planned Features

* Rendering system
* Input handling

* Entity management
